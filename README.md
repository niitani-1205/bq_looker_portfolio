# bq_looker_portfolio
SQL・BigQuery・Looker Studio を用いたデータ分析ポートフォリオです。
BigQuery の公開サンプルデータである『StackOverflow』を使用しました。

『StackOverflow』とは、世界中のエンジニアがプログラミングに関する質問と回答を投稿する【Q&A サイト】です。

本分析では、その公開データを用いて以下を年度ごとに可視化・分析しています：
  ・質問数・回答数の推移
  ・質問から回答までに要した時間（1時間以内・24時間以内）の回答率
  ・質問から1年間、回答がつかなかった割合（未解決率）

これらの指標を BigQuery で集計し、Looker Studio を用いてダッシュボードとして可視化しました。
